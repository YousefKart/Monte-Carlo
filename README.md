# Monte Carlo Engine

🚧 **Work in Progress**

This project is currently under active development. Features, structure, and documentation may change frequently as the engine evolves.

---

## Build Instructions

### Requirements

- CMake ≥ 3.20
- C++20 compatible compiler
- Git

### Clone the repository

```bash
git clone https://github.com/YousefKart/Monte-Carlo.git
cd Monte-Carlo
```

### Configure

```bash
cmake -B build
```

### Build

```bash
cmake --build build --config Debug
```

### Run

```bash
./build/Debug/MonteCarlo
```

Alternatively, if using the provided CMake run target:

```bash
cmake --build build --target run
```
